# Ubuntu安装nginx

# 1.下载文件

前往http://nginx.org/download/ 选择所需的版本

wget  http://nginx.org/download/nginx-1.4.2.tar.gz

# 2.解压文件

tar -zxvf nginx-1.4.2.tar.gz nginx-1.4.2

# 3.下载所需要的依赖库

apt-get install pcre
apt-get install pcre-devel
apt-get install libpcre3 libpcre3-dev
apt-get install zlib
apt-get install zlib-devel
apt-get install zlib1g-dev

# 4.运行configure配置

cd /usr/local/nginx-1.4.2 && ./configure --prefix=/usr/local/nginx1.4.2

如果不报错，说明通过了环境配置的检查。

# 5.编译安装

执行完第四步时，当前目录是/usr/local/nginx-1.4.2。在当前目录下，执行

make && make install

# 6.启动

在/usr/local/nginx1.4.2目录下，有四个文件夹：conf配置文件、html网页文件、logs日志文件、sbin二进制文件。

# 启动命令

sbin/nginx

# 关闭命令

sbin/nginx -s stop

# 重启命令

sbin/nginx -s reload

# 7.检查是否启动

浏览器访问地址：

http://ip:80

看到欢迎页面即可
